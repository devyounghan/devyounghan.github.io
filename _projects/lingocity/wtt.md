### 소개
{:.my-heading-circle-1}

- 인게임에 별도 작업용·편의 기능들을 추가하여 런타임 중 작업을 지원하는 개발 보조 툴
- Windows 플랫폼
- 비개발 직군 팀원 대상
- 팀원 요청 반영한 기능 추가
- 편의성 중심의 간편한 기획·검증 작업 지원


### 개발 목적
{:.my-heading-circle-1}

- 기존 모바일 앱의 복잡한 빌드 프로세스와 그로 인한 중간 산출물 확인의 불편함 해소
- 모바일 기기와 PC를 오가며 작업하는 업무 비효율 개선
- 접근성 강화
- 기획 단계 반복 작업 효율화 및 프로토타입 제작 지원
- 팀원 간 소통 효율 증대

⇒ 개발 효율성 향상
{:.lead}

### 특징
{:.my-heading-circle-1}

- IMGUI, UGUI를 함께 활용한 UI 구성
- 오픈 소스, 애셋 기능 추가 활용
- 툴 전용 모드 추가
- 링고시티 기본 플레이 모드, 작업 모드 분리
- 에디터와 유사한 작업 환경 지향


### 주요 기능
{:.my-heading-circle-1}

- 자유 시점
- NPC 생성 및 수정
- 씬 내 오브젝트 수정
- 씬 자유 이동
- BGM 교체(로컬 파일)
- 기타 치트 기능
